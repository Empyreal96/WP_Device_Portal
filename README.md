# WP_Device_Portal
Small toolkit for utilising features from Windows Device Portal on Windows 10 Mobile

## Features:
Device Info:
 - Network Adapter Info
 - WiFi IP Info

Applications:
- Install
- Uninstall
- Launch
- Information

Services:
- Running Services


### Notes
- This is heavily based of the WindowsDevicePortalWrapper library and UWP sample.
